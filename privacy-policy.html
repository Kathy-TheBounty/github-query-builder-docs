Privacy Policy

GitHub Query Builder For Recruiters  ·  Last updated: April 2026

This extension does not collect, store, or transmit any personal data.

Data Usage

  • Search queries you build are sent directly to the public GitHub Search API (api.github.com) to count results. No query is stored on any server.
  • Saved searches and language preferences are stored locally in your browser via chrome.storage. This data never leaves your device.
  • No analytics, no tracking, no third-party services.

Permissions Used

  • storage — to save your searches and language preference locally in your browser.
  • tabs — to open GitHub search results in a new tab when you click "Open ↗".
  • clipboardWrite — to copy generated queries to your clipboard.
  • api.github.com — to count the number of results for each generated query using GitHub's public API.

Developer Access

The developer does not have access to any data entered or generated by this extension. All data stays on your device.

Third-Party Services

This extension makes requests to api.github.com, GitHub's public Search API. No authentication token is required. These requests are subject to GitHub's Privacy Policy.

Changes to This Policy

If this policy is updated, the "Last updated" date at the top of this page will be revised. Continued use of the extension after any changes constitutes acceptance of the new policy.
